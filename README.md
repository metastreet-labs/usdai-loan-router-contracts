# USDai Loan Router Contracts

## Usage

Clone:

```shell
$ git clone --recursive https://github.com/metastreet-labs/usdai-loan-router-contracts.git
```

Build:

```shell
$ forge build
```

Test:

```shell
$ forge test
```

## Update Submodules

```shell
$ git submodule deinit --force .
$ git submodule update --init --recursive
```
